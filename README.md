# 🧠 AI Image Detector

A web-based application that detects whether an image is **AI-generated or real** using advanced AI APIs.

---

## 🚀 Features

- 📸 Upload any image  
- 🧠 Detect AI-generated vs real images  
- 📊 Confidence score output  
- 🎨 Clean and responsive UI  
- ⚡ Real-time processing using API  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Backend:** Python (Flask)  
- **API:** Sightengine AI Detection API
